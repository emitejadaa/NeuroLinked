Pipeline robusto

Preprocesamiento MNE, estandarización por canal y ventana; modelos optimizados para EEG (EEGNet/Conv-RNN/Transformers).

Integración instantánea

Backend Express + Python. Endpoint /predict con multipart para EDF y modelo opcional, salida JSON normalizada.

Automatización IoT

Conectores a domótica (opcional). Activá escenas, accesos y dispositivos cuando el modelo detecta la intención.

Privacidad primero

Ejecución local o en tu nube. No almacenamos EEGs: los archivos temporales se purgan luego del proceso.

Visualizaciones claras

El predictor expone plot_url del ensayo analizado para inspección y validación rápida.

Extensible

Entrená y versioná tus propios modelos. Cambiá el DEFAULT_MODEL sin tocar el front.
